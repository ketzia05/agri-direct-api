# AgriDirect - Farmer to Consumer Marketplace API

**Project Overview**

AgriDirect is a backend system designed to enable farmers to directly sell their products to consumers, eliminating intermediaries and improving pricing transparency.
This project is being developed in multiple phases to demonstrate backend development skills and system design understanding.

**Project Phases**
- Core Java (Business logic implementation using classes and collections)
- JDBC (Database integration using MySQL)
- Spring Boot (RESTful API development)

**Features**
- Farmer product listing
- Buyer product browsing
- Order management system

**Tech Stack**
- Java
- Spring Boot
- MySQL
- JPA/Hibernate
- Postman
